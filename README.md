# 🚀 PromptPilot

> An intelligent local AI platform that automatically routes prompts to the most suitable Large Language Model (LLM).

> ⚠️ This project is currently under active development.

## 📖 About

PromptPilot is a local AI router built with Python, FastAPI, Ollama and Open WebUI.

Instead of sending every prompt to a single model, PromptPilot analyzes the user's request and automatically selects the most suitable model for the task.

The goal is to provide a fast, private and extensible local AI experience.

## ✨ Current Features

- 🤖 Automatic model routing
- 💻 Coding model support
- 🧠 Reasoning model support
- ✍️ Writing model support
- 💬 Shared conversation history
- ⚡ FastAPI backend
- 🏠 Local execution using Ollama

## 🛠 Tech Stack

- Python
- FastAPI
- Ollama
- Open WebUI
